# T-banner

Herramienta De Pesonalizacion Con Python2

# INSTALACIÓN:

1. apt update && apt upgrade 
2. pkg install git 
3. git clone https://github.com/Josezpt/T-banner
4. cd T-banner 
5. chmod +x *
6. ./start 

# Eliminación de banner
1. cd T-banner 
2. ./start --Delete
